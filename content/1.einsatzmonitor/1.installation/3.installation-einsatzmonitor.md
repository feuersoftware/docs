# Installation EinsatzMonitor

![EinsatzMonitor](/images/einsatzmonitor/EinsatzMonitor_Download.png)

Diese Anleitung führt Sie durch die Installation des EinsatzMonitors auf Ihrem Windows-System.

## Vorbereitungen

### Downloadserver zu Vertrauenswürdigen Sites hinzufügen

Um Probleme beim Download und Installation von Softwareupdates des EinsatzMonitors zu verhindern, ist es sinnvoll die URL des Downloadservers zu den vertrauenswürdigen Sites im Internet Explorer hinzuzufügen. Die Einstellung verhindert, dass es zu Sicherheitswarnungen bei der automatischen Installation von Updates (z.B. nach einem automatischen Neustart) kommt.

**URL zu Vertrauenswürdige Sites hinzufügen:**

```
https://feuersoftware.com/Download/5.1/
```

So fügen Sie die URL hinzu:

1. Öffnen Sie die **Systemsteuerung** unter Windows
2. Klicken Sie auf **Internetoptionen**
3. Wählen Sie den Reiter **Sicherheit**
4. Wählen Sie **Vertrauenswürdige Sites**
5. Klicken Sie auf den Button **Sites**
6. Fügen Sie die URL hinzu

![Vertrauenswürdige Sites](/images/einsatzmonitor/EinsatzMonitor_TrustedSites.png)

::callout{type="info"}
Sollte Ihr Computer Mitglied einer zentral verwalteten Domäne sein, kann das Hinzufügen von URLs durch die IT-Abteilung gesperrt worden sein. Wenden Sie sich hierzu an Ihre zuständige IT-Abteilung.
::

## Download

Der EinsatzMonitor kann über folgenden Link heruntergeladen werden:

:button-link[Download EinsatzMonitor]{href="https://feuersoftware.com/Download/5.1/publish.htm" external}

Klicken Sie auf **Installieren** um die Installationsdateien herunterzuladen. Es wird die Installationsdatei `setup.exe` heruntergeladen.

![Download](/images/einsatzmonitor/EinsatzMonitor_Download.png)

## Installation

1. Führen Sie die heruntergeladene `setup.exe` aus
2. Klicken Sie auf **Installieren**
3. Das Setup lädt die erforderlichen Dateien herunter und installiert den EinsatzMonitor automatisch

![Installation Schritt 1](/images/einsatzmonitor/EinsatzMonitor_Installation01.png)

![Installation Schritt 2](/images/einsatzmonitor/EinsatzMonitor_Installation02.png)

::callout{type="warning"}
Während der Installation ist eine Internetverbindung erforderlich, da die benötigten Komponenten während der Installation heruntergeladen werden.
::

Der EinsatzMonitor wird nach folgendem Pfad installiert:

```
C:\Users\<Username>\Einsatz_Monitor
```

::callout{type="info"}
Der Installationspfad kann nicht angepasst werden.
::

## Erster Start

Nach erfolgreicher Installation erscheint ein Desktop-Icon "Einsatz Monitor 5.1" auf dem Desktop.

![Desktop Icon](/images/einsatzmonitor/EinsatzMonitor_DesktopIcon.png)

1. Starten Sie den EinsatzMonitor durch einen **Doppelklick** auf das Desktop-Icon
2. Beim ersten Start müssen Sie die **Nutzungsbedingungen** akzeptieren
3. Das Fenster schließt sich nach dem Bestätigen automatisch

![AGBs](/images/einsatzmonitor/EinsatzMonitor_AGBs.png)

Der EinsatzMonitor startet und zeigt eine kurze Einweisung an. Beginnen Sie nun mit der Einrichtung.

## Ausnahme im Virenscanner hinzufügen

Wir empfehlen die SQLite-Datenbank des EinsatzMonitors als Ausnahme im Virenscanner zu konfigurieren. Während des Scans des Virenscanners ist diese blockiert und der EinsatzMonitor kann nicht mehr darauf zugreifen. Dies kann zu unerwarteten Abstürzen führen.

### Ausschluss im Windows Defender hinzufügen

1. Öffnen Sie **Windows-Sicherheit**
2. Klicken Sie auf **Viren- & Bedrohungsschutz**
3. Wechseln Sie zu den **Einstellungen für Viren- & Bedrohungsschutz**
4. Wählen Sie **Ausschlüsse hinzufügen oder entfernen**
5. Fügen Sie folgende Datei als Ausschluss hinzu:

```
C:\Users\<Username>\Einsatz_Monitor\EinsatzMonitor.sqlite
```

![Windows Defender Ausschluss](/images/einsatzmonitor/Einsatzmonitor_Defender_Exclusion.png)

::callout{type="tip"}
Wie die Ausnahme bei anderen Anti-Virus-Programmen konfiguriert wird, hängt vom installierten Programm ab. Informationen hierzu finden Sie in der Dokumentation Ihres Anti-Viren-Herstellers.
::
