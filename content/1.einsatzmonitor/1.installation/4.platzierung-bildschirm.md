# Platzierung des Bildschirms in der Wache

Die richtige Platzierung des EinsatzMonitor-Bildschirms ist entscheidend für eine schnelle und effektive Informationsaufnahme bei einem Alarm.

## Grundprinzipien

### Sichtbarkeit

Der Bildschirm sollte von allen relevanten Bereichen der Wache gut sichtbar sein:

- **Aufenthaltsraum** - Wo sich die Einsatzkräfte hauptsächlich aufhalten
- **Umkleidebereich** - Falls vorhanden und in der Nähe
- **Fahrzeughalle** - Optional als zusätzlicher Monitor

### Blickrichtung

- Platzieren Sie den Monitor auf **Augenhöhe** oder leicht darüber
- Vermeiden Sie extreme Blickwinkel (über 45°)
- Berücksichtigen Sie verschiedene Betrachtungsabstände

## Optimale Positionen

### Im Aufenthaltsraum

```
┌─────────────────────────────────────┐
│                                     │
│    ┌───────────┐                    │
│    │  Monitor  │    ← Gut sichtbar  │
│    └───────────┘      von allen     │
│                       Sitzplätzen   │
│   ╭───╮  ╭───╮  ╭───╮              │
│   │   │  │   │  │   │  Sitzbereich │
│   ╰───╯  ╰───╯  ╰───╯              │
│                                     │
└─────────────────────────────────────┘
```

### Im Flurbereich

Bei Platzierung im Flur zum Umkleideraum oder zur Fahrzeughalle:

- Monitor an der Wand in Laufrichtung
- Keine Hindernisse im Sichtfeld
- Ausreichend Abstand zum Durchgangsbereich

## Vermeiden Sie

::callout{type="warning"}
**Ungünstige Platzierungen:**

- **Direkt gegenüber von Fenstern** - Blendung und Reflexionen
- **In dunklen Ecken** - Schlechte Sichtbarkeit
- **Zu hoch** - Unbequemer Blickwinkel
- **Hinter Hindernissen** - Eingeschränkte Sicht
::

## Beleuchtung

### Umgebungslicht

- Vermeiden Sie direkte Sonneneinstrahlung auf den Bildschirm
- Nutzen Sie Vorhänge oder Jalousien bei Bedarf
- Achten Sie auf gleichmäßige Raumbeleuchtung

### Bildschirmhelligkeit

- Passen Sie die Helligkeit an die Umgebung an
- Höhere Helligkeit bei hellem Umgebungslicht
- Reduzierte Helligkeit bei Nacht (automatische Anpassung im EinsatzMonitor möglich)

## Empfohlene Betrachtungsabstände

| Bildschirmgröße | Minimaler Abstand | Optimaler Abstand |
|-----------------|-------------------|-------------------|
| 32 Zoll | 1,2 m | 2,0 - 2,5 m |
| 43 Zoll | 1,6 m | 2,5 - 3,5 m |
| 55 Zoll | 2,1 m | 3,5 - 4,5 m |
| 65 Zoll | 2,5 m | 4,0 - 5,5 m |

## Multi-Monitor-Setup

Bei größeren Wachen kann ein Multi-Monitor-Setup sinnvoll sein:

### Mögliche Konfigurationen

1. **Hauptmonitor + Nebenmonitor**
   - Aufenthaltsraum: Vollständige Ansicht
   - Fahrzeughalle: Einsatzkarte und Stichworte

2. **Gespiegelte Anzeige**
   - Mehrere Monitore zeigen identische Inhalte
   - Einfache Konfiguration

3. **Erweiterte Anzeige**
   - Verschiedene Informationen auf verschiedenen Monitoren
   - Erfordert entsprechende Konfiguration

## Kabelführung

::callout{type="tip"}
**Tipps für eine saubere Installation:**

- Verwenden Sie Kabelkanäle für eine ordentliche Optik
- Planen Sie ausreichend Kabellänge ein
- Nutzen Sie HDMI-Kabel mit ausreichender Länge (bis 15m mit gutem Kabel, darüber aktive Kabel oder Extender)
- LAN-Kabel für stabile Netzwerkverbindung
::

## Checkliste

- [ ] Monitor ist von allen relevanten Bereichen sichtbar
- [ ] Keine störenden Reflexionen oder Blendung
- [ ] Betrachtungsabstand ist angemessen
- [ ] Blickwinkel ist komfortabel
- [ ] Kabel sind sauber verlegt
- [ ] Netzwerk- und Stromversorgung sind stabil
