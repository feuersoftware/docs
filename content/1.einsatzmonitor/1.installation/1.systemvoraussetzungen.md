# Systemvoraussetzungen

Der EinsatzMonitor stellt bestimmte Anforderungen an die Hardware und Software des Systems. Beachten Sie diese Voraussetzungen, um einen reibungslosen Betrieb zu gewährleisten.

## Betriebssystem

| Anforderung | Beschreibung |
|-------------|--------------|
| **Betriebssystem** | Windows 10 oder Windows 11 (64-Bit) |
| **.NET Framework** | .NET Framework 4.7.2 oder höher |

## Hardware-Anforderungen

### Mindestanforderungen

| Komponente | Anforderung |
|------------|-------------|
| **Prozessor** | Dual-Core CPU, 2.0 GHz |
| **Arbeitsspeicher** | 4 GB RAM |
| **Festplatte** | 500 MB freier Speicherplatz |
| **Grafik** | DirectX 9 kompatible Grafikkarte |
| **Bildschirm** | 1920x1080 (Full HD) |
| **Netzwerk** | Internetverbindung erforderlich |

### Empfohlene Anforderungen

| Komponente | Empfehlung |
|------------|------------|
| **Prozessor** | Quad-Core CPU, 2.5 GHz oder höher |
| **Arbeitsspeicher** | 8 GB RAM |
| **Festplatte** | SSD mit mindestens 1 GB freiem Speicherplatz |
| **Grafik** | Dedizierte Grafikkarte mit 1 GB VRAM |
| **Bildschirm** | 1920x1080 oder 3840x2160 (4K) |
| **Netzwerk** | Stabile Breitband-Internetverbindung |

## Netzwerk-Anforderungen

Der EinsatzMonitor benötigt Zugriff auf folgende Dienste:

- **Connect Portal** - `connect.feuersoftware.com` (Port 443/HTTPS)
- **Update Server** - `feuersoftware.com` (Port 443/HTTPS)
- **Kartendienste** - Zugriff auf OpenStreetMap oder andere konfigurierte Kartendienste

::callout{type="info"}
Stellen Sie sicher, dass die Firewall den Zugriff auf diese Dienste erlaubt.
::

## Hinweise

::callout{type="warning"}
Der EinsatzMonitor ist eine Windows-Anwendung und läuft nicht auf macOS oder Linux.
::

Für den 24/7-Betrieb empfehlen wir industrietaugliche Hardware, die für den Dauerbetrieb ausgelegt ist.
