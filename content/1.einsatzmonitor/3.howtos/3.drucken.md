# Drucken einrichten

Der EinsatzMonitor kann bei Alarmeingang automatisch Einsatzaufträge auf einem konfigurierten Drucker ausdrucken. Das Drucken wird über das **Drucken-Plugin** im [Alarmablauf](../2.einstellungen/6.alarmablauf) eingerichtet.

## Voraussetzungen

- Ein unter Windows installierter und funktionsfähiger Drucker
- Ein konfiguriertes Drucktemplate (Dokumentvorlage)

## Einrichtung

1. Einstellungen des EinsatzMonitors öffnen
2. Zu **Alarmablauf** navigieren
3. Das Plugin **„Drucken"** in den Alarmablauf einfügen
4. Folgende Parameter konfigurieren:

| Parameter | Beschreibung |
|---|---|
| **Druckername** | Drucker auswählen, an den der Ausdruck gesendet wird |
| **Anzahl** | Anzahl der Ausdrucke oder **AAO** (Anzahl richtet sich automatisch nach den Fahrzeugen in der AAO) |
| **Template** | Dokumentvorlage für das Drucklayout auswählen |
