Diese Anleitung zeigt Ihnen, wie Sie den EinsatzMonitor mit dem Feuersoftware Connect-Portal verbinden.

## Voraussetzungen

- [ ] Sie haben Zugang zum [Connect-Portal](https://connect.feuersoftware.com)
- [ ] Der EinsatzMonitor ist installiert und gestartet
- [ ] Eine Internetverbindung besteht

## Schritt 1: Connect Account anlegen

1. Melden Sie sich im [Connect-Portal](https://connect.feuersoftware.com) an
2. Erstellen Sie unter **Geräte** einen neuen "Einsatz Monitor Account"
3. Sie erhalten Benutzernamen und Passwort per E-Mail

## Schritt 2: Zugangsdaten eintragen

1. Öffnen Sie im EinsatzMonitor die **Einstellungen** (F10)
2. Navigieren Sie zu **Connect Verbindung**
3. Tragen Sie Benutzernamen und Passwort ein
4. Klicken Sie auf **Verbindung testen**

::callout{type="info"}
Erst nach einem erfolgreichen Test werden weitere Schaltflächen wie „Synchronisation" und „Wartung" freigeschaltet.
::

## Schritt 3: Verbindung aktivieren

Setzen Sie den Schalter **Connect aktiv** auf aktiv.

::callout{type="warning"}
Änderungen erfordern einen **Neustart** des EinsatzMonitors.
::

## Fehlerbehebung

### Verbindung fehlgeschlagen

1. Prüfen Sie die Internetverbindung
2. Stellen Sie sicher, dass die Firewall `connect.feuersoftware.com` nicht blockiert

```bash
# Test der Erreichbarkeit (in CMD/PowerShell)
ping connect.feuersoftware.com
```

3. Bei Verbindungsproblemen: Neustart des EinsatzMonitors nach Änderung der Connect-Einstellungen
