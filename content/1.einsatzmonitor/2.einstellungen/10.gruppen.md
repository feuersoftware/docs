# Gruppen

![EinsatzMonitor Gruppen](/images/einsatzmonitor/EinsatzMonitor_Gruppen.png)

In den Einstellungen „Gruppen" können lokale Benutzer zu Gruppen zusammengefasst und der Alarmtext für Zusatzalarmierungen definiert werden.

::callout{type="info"}
Sofern Sie **FeuerSoftware Connect** einsetzen, muss hier nichts konfiguriert werden. Die Erstellung und Zuordnung von Benutzern erfolgt über Connect.
::

## Felder

| Feld | Beschreibung |
|---|---|
| **Gruppen** | Liste aller Gruppen; Anklicken zeigt das Gruppendetail |
| **Gruppenname** | Name der Gruppe (z. B. „Führungsgruppe") |
| **Alarmtext** | Nachricht, die bei Alarm an die Gruppe gesendet wird |
| **Variablen** | Verfügbare Variablen für den Alarmtext |
| **Benutzer** | Einzelne Profile, die der Gruppe zugeordnet werden |
| **Hinzufügen** | Neue Gruppe anlegen |
| **Speichern** | Änderungen der Gruppe speichern |
| **Löschen** | Gruppe löschen |
