Die Karteneinstellungen ermöglichen die Anpassung der Kartendarstellung in der Einsatzansicht.

## Zugriff

Einstellungen → Karten

## Standardkarte

Die Standardkarte wird auf der linken Seite angezeigt und visualisiert den Einsatzort.

### Einstellungen

- **Provider** - Auswahl des Kartenproviders (Google, Bing, OpenStreetMap)
- **Marker** - Verschiedene Markierungsmöglichkeiten auswählbar
- **Zoom** - Einstellbar, empfohlen ist eine Zoomstufe von etwa 17 für gute Darstellung
- **Breite in %** - Breite der Standardkarte im Verhältnis zur Routenkarte

## Routenkarte

Zeigt die Route zwischen Unterkunft und Einsatzort rechts an.

### Einstellungen

- **Routenanzeige** - Aktivieren, wenn die Routenkarte zusätzlich angezeigt werden soll
- **Helikoptermodus** - Wenn aktiviert, werden Karte, Kilometer und Zeitangaben auf den Einsatz mit Rettungshubschraubern optimiert
- **Provider** - Verschiedene Kartenprovider zur Auswahl

::callout{type="info"}
Die Karten wechseln bei neuen Daten automatisch innerhalb von ca. 15 Sekunden.
::

## Hydrantenkarte (Wasserkarte.info)

Ermöglicht die Anzeige von Hydranten über https://wasserkarte.info (benötigt eigenen Account und API Key).

### Optionen

- **Hydranten anzeigen** - Aktiviert die Hydrantenanzeige
- **Zugriffsschlüssel (API Key)** - API Key von Wasserkarte.info
- **Radius (x km)** - Suchradius für Hydranten
- **Anzahl anzuzeigender Hydranten** - Maximale Anzahl

## Eigene Google API Keys

Eigene Google Maps API Keys können hinterlegt werden, falls Routendarstellung fehlschlägt oder Limits erreicht werden.

::callout{type="tip"}
Es kann sinnvoll sein, die lokalen Gegebenheiten im Einsatzgebiet (z.B. Hydranten, Rettungspunkte im Wald, Verkehrsbehinderungen, Straßensperrungen) für die optimale Nutzung darzustellen.
::
