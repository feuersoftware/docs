# Karteneinstellungen

![EinsatzMonitor Karte](/images/einsatzmonitor/EinsatzMonitor_Karten.png)

Die Kartenanzeige ist ein zentrales Element des EinsatzMonitors und zeigt den Einsatzort sowie die Route dorthin an.

## Kartenanbieter

Der EinsatzMonitor unterstützt verschiedene Kartenanbieter:

| Anbieter          | Beschreibung                        |
| ----------------- | ----------------------------------- |
| **OpenStreetMap** | Kostenlos, Community-basiert        |
| **Google Maps**   | Detailliert, erfordert API-Key      |
| **HERE Maps**     | Professionell, erfordert API-Key    |
| **Bing Maps**     | Microsoft-Dienst, erfordert API-Key |

### OpenStreetMap (Standard)

OpenStreetMap ist der Standard-Kartenanbieter und erfordert keine zusätzliche Konfiguration.

### Google Maps einrichten

1. Erstellen Sie einen [Google Cloud Account](https://cloud.google.com)
2. Aktivieren Sie die **Maps JavaScript API**
3. Erstellen Sie einen **API-Key**
4. Tragen Sie den API-Key in den Einstellungen ein

::callout{type="info"}
Google Maps bietet ein monatliches Freikontingent, das für die meisten Feuerwehren ausreichend ist.
::

## Kartendarstellung

### Zoom-Level

| Einstellung       | Beschreibung                        |
| ----------------- | ----------------------------------- |
| **Standard-Zoom** | Zoom-Level im Standby               |
| **Alarm-Zoom**    | Zoom-Level bei Alarm                |
| **Auto-Zoom**     | Automatische Anpassung an die Route |

### Kartentyp

- **Straßenkarte** - Klassische Kartenansicht
- **Satellit** - Luftbildaufnahmen
- **Hybrid** - Kombination aus beidem
- **Gelände** - Topografische Darstellung

## Einsatzort

### Markierung

Der Einsatzort wird standardmäßig mit einem roten Marker gekennzeichnet. Sie können anpassen:

- **Marker-Farbe** - Farbe des Einsatzort-Markers
- **Marker-Größe** - Größe des Markers
- **Animation** - Pulsierender Marker für bessere Sichtbarkeit

### Umkreis anzeigen

Zeigen Sie einen Radius um den Einsatzort an:

- **100m Radius** - Für präzise Lokalisierung
- **500m Radius** - Für Umgebungsübersicht
- **Benutzerdefiniert** - Eigener Radius

## Route

### Routenberechnung

| Option                      | Beschreibung                      |
| --------------------------- | --------------------------------- |
| **Startpunkt**              | Adresse der Feuerwache            |
| **Routentyp**               | Schnellste / Kürzeste Route       |
| **Verkehr berücksichtigen** | Aktuelle Verkehrslage einbeziehen |
| **Mautstraßen vermeiden**   | Mautpflichtige Straßen umfahren   |

### Anzeige

- **Routenlinie** - Farbe und Stärke der Routenlinie
- **Fahrzeit** - Geschätzte Anfahrtszeit
- **Entfernung** - Distanz zum Einsatzort

## Standort der Wache

Tragen Sie die exakten Koordinaten Ihrer Wache ein:

1. Öffnen Sie **Einstellungen** → **Karte** → **Standort**
2. Geben Sie die **Adresse** ein oder
3. Klicken Sie auf der Karte auf den Standort
4. Die **Koordinaten** werden automatisch übernommen

```text
Beispiel:
Adresse: Musterstraße 123, 12345 Musterstadt
Koordinaten: 51.1657° N, 10.4515° E
```

## Zusätzliche Kartenelemente

### Hydranten anzeigen

Integration mit Wasserkarte.info:

1. Aktivieren Sie die **Hydrantenanzeige**
2. Verbinden Sie Ihren Wasserkarte.info-Account
3. Hydranten werden automatisch auf der Karte angezeigt

### POIs (Points of Interest)

Zeigen Sie wichtige Punkte auf der Karte an:

- **Krankenhäuser**
- **Rettungswachen**
- **Gefahrstoffbetriebe**
- **Eigene POIs** (manuell hinzufügen)

## Offline-Karten

Für Situationen ohne Internetverbindung:

1. Navigieren Sie zu **Einstellungen** → **Karte** → **Offline**
2. Wählen Sie den **Bereich** zum Herunterladen
3. Klicken Sie auf **Herunterladen**

::callout{type="warning"}
Offline-Karten benötigen zusätzlichen Speicherplatz und müssen regelmäßig aktualisiert werden.
::

## Performance-Tipps

- Verwenden Sie **Vektor-Karten** für schnellere Darstellung
- Reduzieren Sie die **Detailstufe** bei langsamen Systemen
- Aktivieren Sie das **Karten-Caching** für schnellere Ladezeiten
