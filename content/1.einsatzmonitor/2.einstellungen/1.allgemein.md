# Allgemeine Einstellungen

![EinsatzMonitor Standby](/images/einsatzmonitor/EinsatzMonitor_Standby.png)

Die allgemeinen Einstellungen bilden die Basis für die Konfiguration des EinsatzMonitors.

## Zugriff

1. Öffnen Sie die Einstellungen über das **Zahnrad-Symbol** oder **F10**
2. Navigieren Sie zu **Allgemein**

## Grundeinstellungen

### Standby-Bildschirm

Konfigurieren Sie, was der EinsatzMonitor im Standby-Modus anzeigt:

| Option | Beschreibung |
|--------|--------------|
| **Uhrzeit** | Große Zeitanzeige |
| **Wetter** | Aktuelle Wetterdaten |
| **Kalender** | Anstehende Termine |
| **News** | Aktuelle Nachrichten |
| **Verfügbarkeit** | Übersicht verfügbarer Kräfte |

### Automatischer Start

- **Mit Windows starten** - EinsatzMonitor startet automatisch beim Windows-Start
- **Vollbildmodus** - Startet direkt im Vollbildmodus
- **Minimiert starten** - Startet minimiert in der Taskleiste

### Aktualisierung

| Einstellung | Beschreibung |
|-------------|--------------|
| **Automatische Updates** | Updates werden automatisch installiert |
| **Update-Intervall** | Prüfhäufigkeit für neue Updates |
| **Beta-Updates** | Optionale Teilnahme am Beta-Programm |

::callout{type="tip"}
Wir empfehlen, automatische Updates aktiviert zu lassen, um immer die neueste Version mit Sicherheitsfixes zu verwenden.
::

## Anzeige

### Bildschirmeinstellungen

- **Helligkeit** - Anpassung der Displayhelligkeit
- **Kontrast** - Kontrasteinstellung
- **Nachtmodus** - Reduzierte Helligkeit nachts

### Zeitgesteuerte Helligkeit

Sie können verschiedene Helligkeitsstufen für verschiedene Tageszeiten konfigurieren:

| Zeitraum | Empfohlene Helligkeit |
|----------|----------------------|
| **Tag** (06:00 - 20:00) | 80-100% |
| **Abend** (20:00 - 22:00) | 60-80% |
| **Nacht** (22:00 - 06:00) | 20-40% |

## Töne und Benachrichtigungen

### Alarmton

- **Alarmton aktivieren** - Akustische Benachrichtigung bei Alarm
- **Lautstärke** - Einstellung der Lautstärke
- **Eigener Ton** - Upload eines eigenen Alarmtons (WAV-Format)

### Testmodus

Für Tests und Vorführungen:

- **Testmodus aktivieren** - Unterdrückt echte Alarme
- **Demo-Modus** - Zeigt Beispieldaten an

## Sprache und Region

| Einstellung | Optionen |
|-------------|----------|
| **Sprache** | Deutsch, Englisch |
| **Datumsformat** | TT.MM.JJJJ, MM/TT/JJJJ |
| **Zeitformat** | 24-Stunden, 12-Stunden |
| **Einheiten** | Metrisch (km), Imperial (Meilen) |

## Tastenkürzel

| Tastenkürzel | Funktion |
|--------------|----------|
| **F10** | Einstellungen öffnen |
| **F11** | Vollbildmodus umschalten |
| **ESC** | Vollbildmodus beenden |
| **F5** | Ansicht aktualisieren |

## Speicherort der Konfiguration

Die Konfiguration wird gespeichert unter:

```
C:\Users\<Username>\Einsatz_Monitor\config.json
```

::callout{type="warning"}
Ändern Sie die Konfigurationsdatei nur manuell, wenn Sie wissen, was Sie tun. Falsche Einstellungen können zu Problemen führen.
::
