# Energieoptionen

![EinsatzMonitor Energieoptionen](/images/einsatzmonitor/EinsatzMonitor_Energieoptionen.png)

In den Energieoptionen kann der EinsatzMonitor zeit- oder ereignisabhängig in den Energiesparmodus versetzt und der Grafikausgang deaktiviert werden.

## Einstellungen

| Einstellung | Beschreibung |
|---|---|
| **Energiesparmodus nie aktivieren** | Energiesparmodus wird nie angewendet |
| **Energiesparmodus bis auf Einsätze aktivieren** | Monitor wechselt immer in den Energiesparmodus – außer bei aktiven Einsätzen |
| **Energiesparmodus in einer bestimmten Zeit aktivieren** | Energiesparmodus wird nur im angegebenen Zeitfenster aktiviert (Von–Bis konfigurierbar) |
| **Energiesparmodus via Lichtsensor** | Monitor wechselt per Lichtsensor in den Energiesparmodus (Lichtsensor von [CLEWARE](http://www.cleware-shop.de)) |
| **Grafikausgang auch deaktivieren** | Deaktiviert zusätzlich den Bildschirm. Erfordert Unterstützung durch Grafikkarte und Monitor (HDMI-Signaldurchleitung) |
| **Lichtsensor / Luminos von CLEWARE** | Lichtsensor auswählen, Lichtstärke auslesen und Schwellenwert per Schieberegler festlegen |
