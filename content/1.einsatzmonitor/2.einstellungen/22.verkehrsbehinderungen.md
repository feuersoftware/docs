# Verkehrsbehinderungen

Im EinsatzMonitor können Verkehrsbehinderungen eingetragen werden, um Einsatzkräfte zu informieren. Sie werden in der **Einsatzansicht auf der Karte** sowie im **Standby als Standbyelement** angezeigt.

## Verkehrsbehinderung anlegen

1. Einstellungen des EinsatzMonitors öffnen
2. Menüpunkt **„Verkehrsbehinderungen"** wählen
3. Informationen eintragen und das rote Kreuz auf die richtige Kartenposition setzen
   - Karte verschieben: rechte Maustaste gedrückt halten
   - Marker hinzufügen: Doppelklick auf das rote Kreuz oder auf das grüne Kreuz links der Karte
   - Marker löschen: rotes X links der Karte
4. Mit **„Verkehrsbehinderung speichern"** bestätigen

![EinsatzMonitor Verkehrsbehinderungen](/images/einsatzmonitor/EM_Verkehrsbehinderung.jpg)

| Feld | Beschreibung |
|---|---|
| **Beschreibung** | Text zur Behinderung (z. B. „Baustelle, Fahrbahn nur einspurig") |
| **Marker** | Position auf der Karte |
| **Startdatum** | Ab wann die Behinderung gilt |
| **Enddatum** | Bis wann die Behinderung gilt |

## Anzeige

### Im Standby

Das Standbyelement **„Verkehrsbehinderungen"** in der Standbyansicht hinzufügen. Die Einträge werden als Text dargestellt.

### In der Einsatzansicht

Die Verkehrsbehinderung wird als Symbol auf der Routenkarte eingeblendet.
