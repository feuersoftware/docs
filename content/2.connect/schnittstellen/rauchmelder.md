# Rauchwarnmelder

FeuerSoftware Connect kann Alarme verschiedener Rauchwarnmelder-Systeme empfangen und direkt als Einsatz weiterverarbeiten. Die unterstützten Integrationen unterscheiden sich in der Einrichtung je nach Hersteller und verwendetem System.

::card-group
  ::card{title="Homematic IP HCU1" icon="i-lucide-flame" to="/connect/schnittstellen/homematic_hcu1"}
  Direkte Alarmweiterleitung über das Connect-Plugin der Homematic IP HCU1 – ohne zusätzliche Middleware.
  ::

  ::card{title="Homematic CCU3" icon="i-lucide-flame" to="/connect/schnittstellen/homematic_ccu3"}
  Alarmweiterleitung über ein HomeMatic Script auf der CCU3 an die öffentliche Connect API.
  ::

  ::card{title="X-Sense" icon="i-lucide-flame" to="/connect/schnittstellen/xsense"}
  Alarmweiterleitung von X-Sense Rauchwarnmeldern über die X-Sense Cloud an Connect.
  ::

  ::card{title="Lupus Electronics" icon="i-lucide-flame" to="/connect/schnittstellen/lupus"}
  Alarmweiterleitung von Lupus Rauchwarnmeldern über die Lupus Cloud an Connect.
  ::
::
