# Anleitung: Kalender-CSV-Import in Feuersoftware Connect

Mit der Kalenderintegration in Feuersoftware Connect können Termine bequem per CSV-Datei importiert werden. Diese Anleitung beschreibt das benötigte Format, alle gültigen Spaltennamen, erlaubte Werte und enthält eine Beispiel-CSV sowie Hinweise zur Fehlervermeidung.

## Anforderungen an die CSV-Datei
- Die Datei **muss im CSV-Format** vorliegen.
- Die **Spaltenreihenfolge ist beliebig**.
- Die **Spaltennamen müssen exakt** so geschrieben sein wie definiert.
- Zeichensatz: **UTF‑8 wird empfohlen**.
- Datumsformat: **TT.MM.JJ** (z. B. `06.01.26`)
- Zeiten als **HH:MM** (z. B. `19:00`)

## Pflichtfelder (müssen vorhanden sein)
| Spaltenname  | Beschreibung |
|--------------|--------------|
| `Subject` | Titel/Betreff des Termins |
| `Start Date` | Startdatum im Format `TT.MM.JJ` |
| `Start Time` | Startzeit im Format `HH:MM` |
| `End Date` | Enddatum im Format `TT.MM.JJ` |
| `End Time` | Endzeit im Format `HH:MM` |

## Optionale Felder (können ergänzt werden)
### `All Day Event`
- Typ: **Boolean**
- Erlaubte Werte: `TRUE`, `FALSE`
- Hinweis: Wenn `TRUE`, müssen **Start Time** und **End Time mindestens `00:00`** sein.

### `Description`
- Freitext

### `Location`
- Freitext

### `Registration Enabled`
- Typ: Boolean  
- Werte: `True`, `False`

### `Registration Maximum`
- Typ: numerisch oder leer  
- Werte:
  - `(leer)` → keine Begrenzung  
  - Zahl ≥ 1

### `Registration Reply Option`
- Typ: Enum  
- Werte:
  - `YesOrNo`
  - `All`

### `Registration Comment Setting`
- Typ: Enum  
- Werte:
  - `NotRequired`
  - `RequiredWhenDeclined`
  - `RequiredWhenDeclinedOrTentative`
  - `AlwaysRequired`

### `Deadline`
- Typ: TimeSpan  
- Gültige Werte:
  - `01:00:00`
  - `02:00:00`
  - `06:00:00`
  - `12:00:00`
  - `1.00:00:00`
  - `2.00:00:00`
  - `3.00:00:00`
  - `7.00:00:00`
  - `14.00:00:00`
  - `30.00:00:00`

Interpretation:
- `01:00:00` → 1 Stunde  
- `1.00:00:00` → 1 Tag  
- `30.00:00:00` → 30 Tage  

## Beispiel-CSV (minimal, nur Pflichtfelder)
```csv
Subject,Start Date,Start Time,End Date,End Time
Monatsübung Gruppe A,06.02.26,19:00,06.02.26,21:00
Atemschutzübung,15.03.26,18:30,15.03.26,21:00
Dienstbesprechung,02.04.26,20:00,02.04.26,21:30
```

## Upload der CSV in Connect
1. Portal öffnen → **Kalender**.
2. **CSV-Import** auswählen.
3. Datei hochladen.
4. System prüft:
   - Pflichtfelder vorhanden?
   - Werte gültig?
   - Format korrekt?
5. Termine werden hinzugefügt.

## Häufige Fehlerquellen & Tipps
- **Spaltenname falsch geschrieben** → Datei wird abgelehnt.  
- **Datumsformat falsch** (z. B. `2026-02-06` statt `06.02.26`).  
- **`All Day Event = TRUE`** → Start-/Endzeit *muss* mindestens `00:00` sein.  
- **Registration Maximum** → Keine 0, nur leer oder ≥1.

## Excel-Vorlage
Eine fertige Excel-Vorlage mit allen Validierungen steht ebenfalls zur Verfügung.
